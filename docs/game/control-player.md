# Control the player
## Move
Use `W`, `A`, `S`, `D` to walk, `SPACE` to jump. `TAB` to toggle fly mode(Just like Minecraft Java Edition).

## Build & Destroy
Left click to destroy(or attack) blocks, right click to build(or use) blocks.

## Others
- `E` - open the bag
- `ESC` - pause the game
- `F1` - show/hide HUD
- `F2` - screenshot
- `F3` - show details
- `F11` - toggle fullscreen
