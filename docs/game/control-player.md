# Control the player

## Move
- `W`, `A`, `S`, `D` - Move
- `SPACE`, `LSHIFT` - Jump or snake
- Quickly press `W`, `SPACE` twice - Toggle run or fly

## Build & Destroy
Left click to destroy (or attack) blocks, right click to build (or use) blocks.

## Others
- `E` - open the bag
- `ESC` - pause the game
- `F1` - show/hide HUD
- `F2` - screenshot
- `F3` - show details
- `F11` - toggle fullscreen
